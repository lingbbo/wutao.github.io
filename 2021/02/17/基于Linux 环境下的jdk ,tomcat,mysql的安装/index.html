基于Linux 环境下的jdk ,tomcat,mysql的安装

一.安装jdk

第一步：解压缩jdk

首先将下载好的jdk存放到 /usr/local中
然后解压缩 tar -zxvf jdk*
删除原压缩包文件rm -f *gz

第二步：配置jdk环境变量

进入文本模式后，在文本最后加入,一定要注意copy配置文件放置的位置（我重装了两次）

按esc :wq保存退出
使刚刚配置的文件生效

输入`java -version 或者输入 java 只要下面没有出现error 或者· not found 之类的错误就配置成功了。

如果没成功 建议重装系统

也可以输入以下命令试一下:输入/bin/vi /etc/profile，试一下能不能把文件改回来就可以了。

二.配置Tomcat

1.首先将下载好的tomcat存放到 /usr/local中
然后解压缩 tar -zxvf apa*
删除原压缩包文件rm -f *gz
2.进入tomcat安装目录里

cd /usr/local/apa*/bin

输入 ./startup.sh
启动tomcat就行了

关闭tomcat 输入

./shutdown.sh
